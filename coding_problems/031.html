Problem 031


Find the inorder successor in a BST


Given a binary search tree and a node, print its inorder successor.
If no inorder successor exists, then print None.

            20
           /   \
          8     22
         /  \
        4    12
            /  \
          10   14

In order successor for 8 is 10.
In order successor for 14 is 20.


Solution



Following observations can be made for the inorder successor:
1. For the greatest element, inorder successor is None.
2. If a node has right subtree, inorder successor is the smallest element in its right subtree.
3. If the node doesn't have a right subtree, the inorder successor is the next greater element.


Algorithm Overview:
1. If the node has a right subtree, find the minimum element in the right subtree.
2. If the node doesn't have a right subtree, traverse the tree from the root and find the next greater element for the node.
