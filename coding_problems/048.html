Problem 048


Painter's partition


We have to paint n boards of length {A1, A2…An}. There are k painters available and each takes 1 unit time to paint 1 unit of the board. The problem is to find the minimum time to get
this job done under the constraints that any painter will only paint continuous sections of boards say board {2, 3, 4} or only board {1} or nothing but not board {2, 4, 5}.

Example:
a = [10, 20, 60, 50, 30, 40]
ans = 90


Solution



We can use binary search and greedy technique to find the minimum time required.
It can be observed that the minimum amount of time possible will be max length of the board.
Similarly, the maximum amount of time can be the sum of all the boards.

The idea is to binary search for the value of the "minimum time" and then check if this will lead to the solution or not.
