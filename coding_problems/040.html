Problem 040


Find the longest substring having unique characters


Given a string, find the longest substring having all unique characters.
Example, for the string "ABBCDEFGHIABCD", the longest substring with all unique characters would be "BCDEFGHIA"


Solution



This problem can be solved in O(N).
We can use a map to store the latest index of the characters that we have seen so far.
For each next character, either we extend the current substring or start a new one after the latest index of the duplicate character.

Algorithm Overview:
1. Maintain a map to store the latest index of the characters.
2. Loop over all the characters.
3. If the character is not in the map, update the map and extend the current substring.
4. If the character is in the map, but the index is smaller than the current start, extend the substring and update the map.
5. If the character is in the map and in the current substring, store the current candidate substring and move the start to the last index of the character + 1.
