Find the minimum cost to connect cities



You are given a set of cities from 0 to N-1, and an adjacency graph which tells which cities are connected to any given city.
We have create roads with minimum cost so that all the cities are accessible.
You have to find out the cost of the road creation.

Example:
cities =
[0, 1, 1, 100, 0],
[1, 0, 1, 0, 0, 0],
[1, 1, 0, 0, 0, 0],
[100, 0, 0, 0, 2, 2],
[0, 0, 0, 2, 0, 2],
[0, 0, 0, 2, 2, 0]
ans = 106



Solution



This is a classic problem to find the minimum spanning tree.
We can use the Prim's algorithm to find the minimum spanning tree for a given graph.

Algorithm Overview:
1. Start with the first node as the starting node.
2. Maintain the cost of all the explored nodes.
3. At each point, select the minimum cost node from the explored nodes and add it to the MST if not already.
4. Do this until all the nodes are in the MST.

As we can see Prim's algorithm is a greedy algorithm.

Complexity:
O(VlogV) where V is the number of nodes in the graph.
