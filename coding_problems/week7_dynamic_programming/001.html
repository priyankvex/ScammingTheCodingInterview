Find the minimum edit distance between words



Given two words A and B, find the minimum number of steps required to convert A to B. (each operation is counted as 1 step.)
You have the following 3 operations permitted on a word:

  • Insert a character
  • Delete a character
  • Replace a character

Example : 
edit distance between
"Heman" and "Hitman" is 2.

  • Operation 1: Replace i with e.
  • Operation 2: Remove t.


Solution



We can solve this problem using Dynamic Programming.
First, just like every problem of dynamic programming, let's try to derive the optimal substructure here.

It can be observed that at any given point we have the following options to deal with the current characters for the strings:
1. Delete the character if they are are equal in one of the string 1 + (i, j-1) or 1 + (i-1. j)
2. Replace the character if not equal in one of the string 1 + (i-1, j-1)
3. Insert the character in one of the string 1 + (i, j-1) or 1 + (i-1. j)

Thus our optimal substructure becomes:

dp[i][j] = {
    if a[i] == a[j]: dp[i-1][j-1],
    else: 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])
}
