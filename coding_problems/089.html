Find the minimum cost path in at most K hops.



We are give a graph of cities connected with flights.
We have to find the best path from source to destination in at most K hops.

Example:

graph = {
    "A": [("B", 1), ("C", 5)],
    "B": [("C", 1)],
    "C": [("D", 1), ("E", 3)],
    "D": [("E", 1)]
}

max_steps = 3
ans  = A -> B -> E (cost 4)

We can't take the path A-> B -> C -> D as it'll cost us 4 hops.



Solution



We can solve this problem using a modified version of BFS.
Idea is to give up searching as soon as we have more than K length path in hand.
