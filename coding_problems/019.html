Problem 019


Merge K Sorted Linked Lists


Given k sorted singly linked lists, write a function to merge all the lists into one sorted singly linked list.



Solution



There are various ways to solve this problem.

Divide and Conquer approach:
One possible way is to pair up the lists and keep merging lists in pairs, till we are left with a single list.
The time complexity of this solution is O(K * NLogN)
Though this doesn't require any extra space.


Using Min-Heap:
Another alternate solution is to use a min-heap of size K.

Algorithm Overview:
1. Maintain a min-heap of size K and insert the first element from each list to the heap.
2. While the heap has elements, pop the top element from the heap.
3. Since this is a min-heap, this will be the minimum element.
4. Append this to the resulting min-heap.
5. Take the next element from the linked list and put it in the heap.
