Problem 008


There is an N by M matrix of zeroes. Given N and M, write a function to count the number of ways of starting at the top-left corner and getting to the bottom-right corner.
You can only move right or down.



For example, given a 2 by 2 matrix, you should return 2, since there are two ways to get to the bottom-right:

  • Right, then
  • Down, then right.

Given a 5 by 5 matrix, there are 70 ways to get to the bottom-right.



Solution



We are given a N X M grid and we have to count the total numbers of ways in which we can start from 0, 0 and reach N-1, M-1.

This can be done efficiently in O(N*M) complexity using extra space.

As it is given that we can move only right and down, numbers of ways to reach square will be :

number of ways to reach the square just above + number of ways to reach the square just to the left.

The overview of the algorithm looks as follows:
1. Create a N X M matrix.
2. Fill the first row and the first column with 1 as there is only 1 way to reach those squares.
3. For the rest of the squares, keep adding the values present in the square on the top of the current square and the square at the left of the current square.
4. The value of the last square is the answer.
