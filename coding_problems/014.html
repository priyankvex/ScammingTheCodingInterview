Problem 014


Find maximum frequency path value for a graph


In a directed graph, each node is assigned an uppercase letter. We define a path's value as the number of the most-frequently-occurring letter along that path. For example, if a path in the graph goes through "ABACA", the value of the path is 3, since there are 3 occurrences of 'A' on the path.

Given a graph with n nodes and m directed edges, return the largest value path of the graph. If the largest value is infinite, then return null.

The graph is represented with a string and an edge list. The i-th character represents the uppercase letter of the i-th node. Each tuple in the edge list (i, j) means there is a directed edge from the i-th node to the j-th node. Self-edges are possible, as well as multi-edges.

The output will be 4, since A will have the maximum frequency out of all the letters which will occur in the path : A -> B -> A -> C -> A



Solution



Let's try to simplify the problem.
For each path of the graph, the value is equal to the frequency of the most frequently occurring letter. Out of all the possible paths, we have to find the maximum value.

Approach 1:

We can do a classic DFS over the graph and go through all the paths possible.
For each path, we keep track of the frequency of each letter.

Out all such paths and their letter frequency count, we find the maximum frequency.

More Efficient Approach:

We can easily notice that if a node has mode than 1 incoming edge, then we will end up doing many repeated computations.

We can cache the result for each of the node as a map that contains the frequency count for that node.
If we ever encounter that node again, we'll use the value from the cache.
