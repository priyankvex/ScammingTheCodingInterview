Problem 041


Find the maximum element in all subarrays of size K


Given an array and an integer k, find the maximum for each and every contiguous subarray of size k.

Example, for the given array
[1, 2, 3, 1, 4, 5, 2, 3, 6] and K = 3
the answer will be
[3, 3, 4, 5, 5, 5, 6]


Solution



A naive solution can be to traverse each window and find the maximum in each window.
The time complexity of this solution is O(NK).

An efficient solution
An efficient solution can be made using the deque structure.
The idea is to keep a deque of the maximum size of K (size of the window).
For each element, remove all the elements from the queue from the left that are out of the window.
From the right, remove all the elements that are smaller than the current element.
Thus at each point, the first element in the queue is the largest element for that sliding window.
