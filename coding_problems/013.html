Problem 013


Unbiased coin toss using a biased coin


Assume you have access to a function toss_biased() which returns 0 or 1 with a probability that's not 50-50 (but also not 0-100 or 100-0).
You do not know the bias of the coin. Write a function to simulate an unbiased coin toss.



Solution



There is a common pattern when we have to simulate a uniform probability using a biased sample space.

We always try to generate a new sample space from the original biased sample space, which as an equal probability.

In this problem, we are given a function that gives H and T with probability X and Y.

Let's consider the case when we toss the coin twice.
The sample space will look like the following:

H H
T H
H T
T T

The probability of T and H coming is (Y * X)/(X+Y)^2

Similarly, the probability of H and T coming is (X * Y)/(X+Y)^2

We can use this to implement an unbiased coin toss function.
