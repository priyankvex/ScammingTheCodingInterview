Problem 050


Maximise j-i such that a[j] > a[i]


Given an array, give the maximum value of j - i, such that A[j] > A[i].

Example,
[9, 2, 3, 4, 5, 6, 7, 8, 18, 0]
ans = 7 (8 - 0)


Solution



The problem can be simplified in a way that, we need to find an index J for I that is farthest from it.
We can keep two arrays, left_min to store minimum element till "i" and right_max to store maximum element on the right.

Using these two arrays, we can find the optimal value for j - i.
