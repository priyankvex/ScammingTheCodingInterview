Problem 012


Find pairs of bishops that attack each other


On our special chessboard, two bishops attack each other if they share the same diagonal. This includes bishops that have another bishop located between them, i.e. bishops can attack through pieces.

You are given N bishops, represented as (row, column) tuples on a M by M chessboard. Write a function to count the number of pairs of bishops that attack each other. The ordering of the pair doesn't matter: (1, 2) is considered the same as (2, 1).

For example, given M = 5 and the list of bishops:

  • (0, 0)
  • (1, 2)
  • (2, 2)
  • (4, 0)

The board would look like this:

[b 0 0 0 0]

[0 0 b 0 0]

[0 0 b 0 0]

[0 0 0 0 0]

[b 0 0 0 0]

You should return 2, since bishops 1 and 3 attack each other, as well as bishops 3 and 4.



Solution



Naive Approach

A naive approach can be for each bishop, iterate over all the bishops and count what pairs attack each other.
The complexity of this approach would be O(N^2)

Efficient Approach

There is a more efficient approach. First, let's simplify the problem.

Two bishops will attack each other i.e lie on the same diagonal if:
1. Sum of their X and Y coordinate is the same, OR
2. The difference of their X and Y coordinate is the same.

We can use this property and store the count of bishops that attack each other in a map.
Once we have the count of bishops that attack each other in place, it's just a matter of getting the combinations count of choosing 2 bishops from K attacking bishops.
