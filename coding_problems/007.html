Problem 007


Implement integer exponentiation. That is, implement the pow(x, y) function, where x and y are integers and returns x^y.

Do this faster than the naive method of repeated multiplication.



For example, pow(2, 10) should return 1024.



Solution



This problem can be solved in a better way than naively multiplying the number N times, which gives the complexity of O(N).

We can use divide and conquer and basic memoization to do this in O(log N).


The overview of the algorithm looks as follows:
- Keep diving the exponent into halves.
- Store the values for the exponents that we have already calculated.
- Use these values to get the value of the original exponent.
