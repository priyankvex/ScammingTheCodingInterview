Problem 042


Find the maximum increasing subsequence sum


Given an array of N integers, find the maximum increasing subsequence sum.
For example, for the given array [6, 5, 4, 1] the answer will be 9 (5 + 4).


Solution



This question is a variant of the famous "longest increasing subsequence" problem, just that the criteria here is not the length but the maximum sum of the subsequence.

This problem can be solved in O(N^2) using dynamic programming.
