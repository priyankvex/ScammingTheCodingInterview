Problem 033


Find two prime numbers with the given sum


Given an even number (greater than 2), return two prime numbers whose sum will be equal to the given number.

Example:
Input: n = 66 Output: 5, 61
Input: n = 9990 Output: 17, 9973


Solution



An observation can be made that we if we are able to generate all the prime numbers that are smaller than the given number N, then this problem can become very easy.

We can use the Sieve of Eratosthenes, to compute all the prime numbers to a given number N.
Once we have the prime numbers we can use the complement method to find the pair.


Algorithm Overview:
1. Create a list of all numbers from 1 to N.
2. Start with p = 2 (the first prime number).
3. Iterate through all the multiples of p and mark them.
4. Find the first number that's bigger than p and is unmarked.
5. Repeat the step 3.
6. Stop when there are no unmarked numbers left that are bigger than p.
7. Use the complement method to find the pair if it exists.
