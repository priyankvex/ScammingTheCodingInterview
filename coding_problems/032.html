Problem 032


Sort an array of 0s, 1s and 2s


Given an array that has only 0, 1, and 2 as it's elements. Sort the array.

Example,
[0, 1, 2, 0, 1, 2] becomes [0, 0, 1, 1, 2, 2]


Solution



This question can be solved using the Dutch National Flag problem.
The main idea is to partition the array into 4 parts:
1. 0 to low always contains 0s
2. low +1 to mid always contains 1s.
3. mid+1 to high -1 is the unknown part.
4. high to N-1 contains 2s.
