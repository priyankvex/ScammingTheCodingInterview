Problem 037


Find the majority element in the array


Given an array find the majority element. A majority element is the element that appears in the array for more than N/2 times.

Ex.
[5, 4, 5, 4, 5, 5] has 5 as the majority element.
[5, 4, 3, 2, 1] has no majority element.


Solution



Basic Approach:
The majority element can be found using extra space and maintaining the count of each element in a map.
The time complexity of this solution will be O(N) and space complexity will be O(N).

Efficient Approach:
An efficient and interesting approach can be to use the Moore's voting algorithm.
The main idea is to counter the occurrence of the selected majority element with every non-majority element. But for this algorithm to be correct, there must be a majority element in the array.

Algorithm Overview:
1. Start with the first element and select it as the majority element.
2. For each element:
3. If the element is equal to the current majority element, then increment the counter.
4. If not then decrement the counter.
5. If the counter becomes zero, then make the current element as the majority element and set the counter to be 1.
6. Verify that the selected element in the end indeed is the majority element (to handle the case where there is no majority element present).
