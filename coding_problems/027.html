Problem 027


Coin game of two


Consider a row of n coins of values v1 . . . vn, where n is even.

We play a game against an opponent by alternating turns. In each turn, a player selects either the first or last coin from the row, removes it from the row permanently, and receives the value of the coin.

Determine the maximum possible amount of money we can definitely win if we move first.

Example:
1. 5, 3, 7, 10 : The user collects maximum value as 15(10 + 5)
2. 8, 15, 3, 7 : The user collects maximum value as 22(7 + 15)


Solution



There are two points to note here:
1. The other player is as smart as the player 1. Thus she'll not make a wrong move.
2. Choosing the maximum value at every point will not work. (Check the example 2).


For a coin array starting at i and ending at j:
We can formalize the following relation.

ans = MAX( A[i] + min(F(i+1, j-1), F(i+2, j)), A[j] + min(F(i, j-1), F(i+1, j-1)) )
Where F(i, j) is the maximum value possible for player 1 for coin array from i to j.

As you can see in the relation, there is some repeated calculation too, thus we can us DP to optimize this too.
