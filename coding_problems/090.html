Solve the 8 puzzle



We all the know the famous 8 puzzle game. Where we have tiles numbered 1 to 8 in a 3 * 3 game board with one blank tile and our job is to arrange the board into a target configuration by swapping tiles with the blank tiles.

We have to implement a 8 puzzle solver to solve the puzzle in minimum moves.
We can assume that for this problem the board is always solvable.

Example:

board = [
    ["1", "2", "3"],
    ["5", "6", "_"],
    ["7", "8", "4"]
]
target_board = [
    ["1", "2", "3"],
    ["5", "8", "6"],
    ["_", "7", "4"]
]

answer = 
 ['1', '2', '3']
 ['5', '6', '_']
 ['7', '8', '4']

 ['1', '2', '3']
 ['5', '_', '6']
 ['7', '8', '4']

 ['1', '2', '3']
 ['5', '8', '6']
 ['7', '_', '4']

 ['1', '2', '3']
 ['5', '8', '6']
 ['_', '7', '4']


Solution



We can start thinking about this problem with standard techniques like DFS and BFS.
Since we have to find the minimum moves, BFS could've worked. But there is one catch, how do we determine if a branch is dead.

Here we'll explore a different approach called as branch and bound. In this approach for every branch that's open we choose a branch that has least cost associated with it.

Cost = Moves so far + Misplaced tiles remaining
