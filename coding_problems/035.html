Problem 035


Find median of a running stream of numbers


Given a stream of numbers, print the running median.

Example,
If the stream looks like [-3, -1, 2, 3]
Then the running median at each point will be [-3, -2, -1, 0.5]


Solution



An observation can be made that, it'll be very efficient to get the mean if we could get the middle values from all the values seen so far efficiently.

We can use two heap data structures to maintain all the smaller and larger elements than the current median.
