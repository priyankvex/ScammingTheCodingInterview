Problem 047


Edit Distance


Given two words A and B, find the minimum number of steps required to convert A to B. (each operation is counted as 1 step.)

You have the following 3 operations permitted on a word:

Insert a character
Delete a character
Replace a character

Example:
ANTMAN can be converted into ANHIMAN by replacing H with T and removing I. Thus, the strings are 2 edit distance away.


Solution



This problem can be solved using Dynamic Programming.
For index i in A and j in B, we can derive the following optimal substructure:

if A[i] == B[j]:
then DP[i][j] = DP[i-1][j-1]
i.e same edit distance as the previous pair of letters.
else:
DP[i][j] = 1 + min(
DP[i-1][j-1]), replacing the characters
DP[i][j-1], removing/inserting chracters
DP[i-1][j], removing/inserting chracters
)
