Problem 025


Divide two numbers without using division, multiply or modulo


Implement division of two positive integers without using the division, multiplication, or modulus operators. Return the quotient as an integer, ignoring the remainder.


Solution



You should start getting a feel that these sort of questions generally involve bit-manipulation.


Naive Approach:
A very basic approach can be to keep adding the divisor till it becomes greater than the dividend.

Bit-Manipulation Approach:
We'll try to form the quotient in the binary form.
This can be done by starting with the most significant bit first and keeping moving towards right.


Algorithm Overview:
1. Iterate from i = 31 to 0
2. If sum_so_far + divisor << i < dividend, then the ith bit is 1 in the quotient.
3. Store the ith bit as one in the quotient.
